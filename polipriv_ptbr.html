Coleta e Uso de Informações

Este aplicativo permite que os usuários personalizem suas próprias preferências, o que significa que pode precisar de permissões específicas para funcionar corretamente. As permissões serão listadas abaixo e elaboradas quanto ao seu uso.

  • INTERNET - Esta permissão é necessária para todas as ações que precisam acessar a internet, como Youtube, Pesquisa de letras, etc;
  • ACCESS_WIFI_STATE - Esta permissão é usada para verificar se o dispositivo tem uma conexão ativa com a Internet;
  • ACCESS_NETWORK_STATE - Esta permissão é usada para verificar se o dispositivo está conectado a uma rede WIFI;
  • READ_EXTERNAL_STORAGE - Esta permissão é usada para ler dados de seu dispositivo, como arquivos de áudio, imagens, vídeos e usá-los para personalizar/reproduzir dentro do aplicativo;
  • WRITE_EXTERNAL_STORAGE - Esta permissão é usada para fazer backup de registros/configurar um servidor Web em seu dispositivo, permitindo a transmissão de conteúdo para sua SmartTV/dispositivo de Cast;
  • WAKE_LOCK - Esta permissão é usada para evitar que o dispositivo hiberne e pare a reprodução repentinamente;
  • FOREGROUND_SERVICE - Esta permissão é usada para criar um serviço de primeiro plano para o reprodutor de áudio;
  • MEDIA_CONTENT_CONTROL - Esta permissão é usada para pausar a reprodução de outros Players quando utilizando este aplicativo;
  • READ_PHONE_STATE - Esta permissão é usada para pausar a reprodução quando você estiver em uma chamada;

As informações que solicitamos são retidas em seu dispositivo e não são coletadas por nós de forma alguma.

O aplicativo usa serviços/bibliotecas de terceiros que podem coletar informações usadas para identificá-lo.

Dados de Registro

Informamos que sempre que você utilizar nosso Serviço, em caso de erro no aplicativo, coletamos dados e informações (por meio de produtos de terceiros) em seu telefone denominado "Dados de Registro". Esses dados de registro podem incluir informações como endereço de protocolo de Internet ("IP") de seus dispositivos, nome do dispositivo, versão do sistema operacional, configuração do aplicativo ao utilizar nosso serviço, a hora e data de uso do serviço e outras estatísticas.

Cookies

Cookies são arquivos com uma pequena quantidade de dados que normalmente é usado como um identificador exclusivo anônimo. Eles são enviados para o seu navegador a partir do site que você visita e são armazenados na memória interna dos seus dispositivos.

Este aplicativo não usa esses "cookies" explicitamente. No entanto, o aplicativo pode usar código de terceiros e bibliotecas que usam "cookies" para coletar informações e melhorar seus serviços. Você tem a opção de aceitar ou recusar esses cookies e saber quando um cookie está sendo enviado para o seu dispositivo. Se você optar por recusar nossos cookies, pode não ser capaz de usar algumas partes deste Serviço.

O aplicativo MuviCast Player usa um navegador interno para realizar pesquisas no Youtube, Google e sites de terceiros(Letras), sendo assim, você está ciente de que Youtube, Google e sites de terceiros (Letras) podem usar cookies nesse navegador para melhorar a experiência do usuário. Não temos afiliação com Google, Youtube ou qualquer outro site acessado através de seu navegador, o app MuviCast Player apenas utiliza seus serviços se comportando como se fosse um navegador normal.

Segurança

Este aplicativo não coleta e processa nenhum de seus dados pessoais, apenas solicita acesso ao Cartão de Memória de seu telefone para personalizar seu aplicativo, como usar suas próprias imagens em seus Álbuns, ler seus arquivos de áudio/vídeos para reproduzi-los dentro do aplicativo. Nenhuma informação sensível como nome, endereço, número de telefone e similares é coletada/compartilhada ou vendida por este aplicativo a terceiros.
